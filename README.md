# Bobby Academy Website

This is a beginner-friendly frontend educational website for the MANTRA 2026 Summer School assignment.

## Website Type
Educational Academy Website

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub
- Netlify

## Pages Included
- Home
- About
- Courses
- Gallery
- Contact

## Features
- Responsive design for mobile, tablet and desktop
- SEO meta tags on every page
- Image alt text
- Mobile navigation menu
- Student Register using JavaScript
- Course cords with responsive layout
- Gallery filter using JavaScript

## How to Run
Open `index.html` in any browser.

## Netlify Deployment Steps
1. Create a GitHub repository.
2. Upload all files from this folder.
3. Login to Netlify.
4. Click "Add new site" > "Import an existing project".
5. Connect GitHub repository.
6. Deploy the website.
